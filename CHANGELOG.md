## [v1.1] - 2026-03-14

### 新增功能

- **双主题系统**：新增风格1（夜间深蓝/金色）与风格2（日间米白/蓝色）两套完整主题，支持一键切换，设置持久化保存到 localStorage
- **搜索历史**：搜索记录自动保存，支持点击历史记录快速重新搜索，支持单条删除和一键清空
- **关于页面**：新增"关于韦达书库"对话框，说明内容来源、参考资源（vedabase.io）和翻译说明
- **书架页面重设计**：全新书架布局，书籍封面卡片化展示，支持双主题样式

### 问题修复

- **[Bug01] 书签页面深色主题样式异常**：修复书签页面在深色主题（风格1）下背景色、文字色、卡片样式不正确的问题，现在深色主题下书签页面完全正常显示
- **[Bug02] 搜索结果页面主题适配**：修复搜索结果在深色主题下部分元素颜色不协调的问题，关键词高亮在两套主题下均正确显示
- **[Bug03] 阅读页面主题切换**：修复 BG/SB 阅读页面切换主题后部分文字颜色未更新的问题
- **[Bug04] 书签精确跳转**：修复从书签列表跳转时跳到章节开头而非具体节的问题，现在点击书签可精确跳转到收藏的具体节（如 BG 8.10 精确跳转到第8章第10节）
- **[Bug05] 底部导航栏 z-index**：调整底部导航栏层级，确保在所有页面上方正确显示

### 优化改进

- 书架页面视觉层次优化，书籍封面、标题、描述信息布局更清晰
- 搜索页面空状态提示优化，引导用户开始搜索
- 阅读页面顶部导航栏在深色主题下对比度提升
- 全局 CSS 变量重构，主题切换更流畅，无闪烁

## [v1.0] - 2026-03-13

### 首次发布

韦达书库网页版首个完整功能版本，基于韦达书库安卓 APP 内容构建。

### 核心功能

- **博伽梵歌（BG）**：完整收录 18 章 657 节，含梵文原文、逐词释义（英文）、中英双语译文、要旨
- **圣典博伽瓦谭（SB）**：完整收录 12 篇 336 章 13,002 节，含梵文原文、逐词释义（英文）、中英双语译文、要旨
- **爱卡达西（Ekadasi）**：完整收录 101 章故事，含破戒时间、节日背景、完整故事内容
- **韦达日历**：2025–2026 年韦达节日日历，含爱卡达西、圣人纪念日等，月历视图+节日详情
- **书签功能**：收藏任意节，支持跨书籍管理，本地持久化存储
- **全文搜索**：支持中文、英文、梵文关键词搜索，结果高亮显示，实时搜索
- **中英双语切换**：译文语言即时切换，逐词释义保持英文
- **字体大小调节**：三档字体大小（小/中/大），设置持久化
- **左右滑动翻页**：移动端手势支持，在章节内节之间滑动翻页
- **响应式设计**：适配手机、平板、桌面端

### 技术实现

- React 18 + TypeScript + Vite 构建
- shadcn/ui + Tailwind CSS 组件库
- 所有内容数据以 JSON 格式静态存储，无需数据库
- localStorage 持久化用户设置（书签、字体、语言等）
