# 2026-03-22 开发会话（第二次）

## 本次修复与新增功能

### Bug 修复

**导航返回逻辑重构**
- 切换 tab 再回书架：始终回到书架首页（home），不再恢复阅读位置
- 从书签进入阅读页 → 返回时回到书签页（通过 `fromBookmark` 标记 + `pop()` 切换 tab）
- 从搜索进入阅读页 → 返回时回到搜索页（通过 `fromSearch` 标记 + `pop()` 切换 tab）
- 爱卡达西阅读位置通过独立 `akadasiSelectedId` 状态维护，切换 tab 后能恢复到上次阅读的那一篇
- 已经在节页时点击书架 tab → 回到书架首页
- 重复点击搜索 tab → 重置搜索页到初始状态

**进度记忆逻辑修正**
- 点击书架上的书籍封面 → 直接进入篇/章列表，不恢复进度
- 进度记忆只用于"继续阅读"按钮

**搜索历史编辑模式**
- 新增"取消"按钮，可以不删除直接退出编辑模式
- 修复"完成"按钮逻辑（选中时删除，未选中时退出编辑模式）

### 新增功能

**翻页区域扩展**
- 点击屏幕左侧或右侧三分之一区域也可以翻页（BGReadPage + SBReadPage）
- 中间三分之一区域不触发翻页（避免误操作）

**目录逻辑修复**
- BG 阅读页目录：显示章列表，点击跳转到该章的节列表页
- SB 阅读页目录：显示篇列表，点击跳转到该篇的章列表页

**日历页遮挡**
- 日历页内容完全被"正在开发中"遮挡，用户无法看到未完成的内容
- 遮挡文案：中英双语，说明功能正在开发中

**搜索高亮颜色**
- 将搜索关键词高亮从金色改为悦人的红色（`#d63031`）
- 深色主题下为浅红色（`#ff7675`）

**搜索结果自动滚动**
- 从搜索进入阅读页后，页面自动滚动到第一个高亮位置
- `SectionContent` 组件新增 `searchKeyword` prop，挂载后自动滚动

**文字替换**
- 全局将"经文"替换为"经典"（包括 Toast 提示、按钮文字等）

**懒加载优化**
- 书架页加载时，1.5 秒后后台预加载 BG 数据和 SB 索引
- SB 阅读页加载完当前篇后，2 秒后后台预加载相邻篇（上一篇和下一篇）
- 通过 `useSBPreload` hook 实现，不影响当前加载性能

**搜索关键词传递**
- 从搜索结果进入阅读页时，搜索关键词通过路由状态传递给阅读页
- 阅读页将关键词传给 SectionContent，实现自动滚动到高亮位置
