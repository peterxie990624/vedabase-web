# 韦达书库 v1.7 - 中英文搜索智能映射与多位置高亮

**日期**：2026-03-29  
**版本**：v1.7  
**主要功能**：中英文搜索智能映射、多位置高亮、自动滑动完善

---

## 任务目标

实现中英文搜索的智能映射和多位置高亮功能，使用户在中文模式下搜索英文关键词时能够自动映射到对应的中文翻译，并在梵文、译文、逐词释义、要旨等多个位置正确高亮关键词。

### 核心需求

1. **中英文映射**：中文模式下搜索英文关键词时，自动映射到中文翻译
2. **多位置高亮**：在梵文、译文、逐词、要旨中都能高亮关键词
3. **自动滑动**：搜索英文或其他位置的关键词时，自动滑动到第一个高亮位置
4. **缺失标记**：中文缺失时显示 "[无中文翻译]" 标记
5. **结果合并**：同一节内多个中文命中合并为一条搜索结果

#### 2. 改进 searchBG 函数

**核心逻辑**：
- 添加 `getMatchInfo()` 辅助函数，确定匹配位置和高亮关键词
- 优先级顺序：词义 > 梵文 > 译文 > 要旨
- 中文模式下搜英文时，自动从中文字段映射（如 `words_zh_fc`）
- 如果中文缺失，标记 `isMissingChinese: true`，并在预览中添加 "[无中文翻译]"

**关键改进**：
```typescript
const getMatchInfo = (section: any): {
  matchLocation: 'sanskrit' | 'translation' | 'wordmeaning' | 'purport';
  highlightKeyword: string;
  isMissingChinese: boolean;
} | null => {
  // 1. 检查中文词义字段
  if (section.words_zh_fc && textContainsKw(section.words_zh_fc)) {
    return { matchLocation: 'wordmeaning', highlightKeyword: keyword, isMissingChinese: false };
  }
  
  // 2. 检查英文词义字段
  if (section.words_en_fc && textContainsKw(section.words_en_fc)) {
    // 中文模式搜英文：尝试从中文词义映射
    if (language === 'zh' && isEnKeyword && section.words_zh_fc) {
      const zhWords = section.words_zh_fc.split(';').map((s: string) => s.trim()).filter(Boolean);
      return { matchLocation: 'wordmeaning', highlightKeyword: zhWords[0] || keyword, isMissingChinese: false };
    }
    // 中文模式搜英文但无中文词义 → 标记缺失
    if (language === 'zh' && isEnKeyword) {
      return { matchLocation: 'wordmeaning', highlightKeyword: keyword, isMissingChinese: true };
    }
  }
  
  // ... 继续检查梵文、译文、要旨
};
```

#### 3. 改进 searchSB 函数

与 searchBG 逻辑一致，支持同样的中英文映射和多位置高亮。

```typescript
function normalizeSanskrit(text: string): string {
  return text
    .replace(/[āàáâãäå]/g, 'a')
    .replace(/[ēèéêë]/g, 'e')
    .replace(/[īìíîï]/g, 'i')
    // ... 其他变音符号
    .toLowerCase();
}
```

#### 3. 改进 highlightText 函数

**核心改进**：
- 使用 `highlightKeyword` 而不是 `searchKeyword`（中文模式下搜英文时已映射）
- 支持梵文规范化匹配（逐字扫描，找到匹配的原始字符串并高亮）
- 支持多位置高亮（只在 `matchLocation` 指定的位置高亮）

```typescript
const highlightText = (html: string, location?: 'sanskrit' | 'translation' | 'wordmeaning' | 'purport'): string => {
  const keyword = highlightKeyword || searchKeyword;
  if (!keyword) return html;
  
  // 如果指定了matchLocation，只在匹配的位置高亮
  if (matchLocation && location && matchLocation !== location) {
    return html;
  }
  
  // 先尝试直接匹配
  const escaped = keyword.replace(/[.*+?^${}()|[\]\\]/g, '\\$&');
  const directRegex = new RegExp(escaped, 'gi');
  if (directRegex.test(html)) {
    return html.replace(new RegExp(escaped, 'gi'), match => 
      `<mark class="search-highlight" style="background:#d4a017;color:#1a1a1a;...">${match}</mark>`
    );
  }
  
  // 梵文规范化匹配：逐字扫描
  const normalizedKw = normalizeSanskrit(keyword);
  let result = '';
  let i = 0;
  const normalizedHtml = normalizeSanskrit(html);
  
  while (i < html.length) {
    if (normalizedHtml.slice(i, i + normalizedKw.length) === normalizedKw) {
      const original = html.slice(i, i + normalizedKw.length);
      result += `<mark class="search-highlight" ...>${original}</mark>`;
      i += normalizedKw.length;
    } else {
      result += html[i];
      i++;
    }
  }
  
  return result === html ? html : result;
};
```

#### 4. 在多个位置应用高亮

- 梵文：`<pre dangerouslySetInnerHTML={{ __html: highlightText(parsedVerse, 'sanskrit') }} />`
- 逐词：`<span dangerouslySetInnerHTML={{ __html: highlightText(pair.sk, 'wordmeaning') }} />`
- 译文：`<div dangerouslySetInnerHTML={{ __html: highlightText(translation, 'translation') }} />`
- 要旨：`<div dangerouslySetInnerHTML={{ __html: highlightText(cleanPurport, 'purport') }} />`

## 工作流程示例

### 场景 1：中文模式下搜索英文关键词 "devotion"

1. **SearchPage**：
   - 用户输入 "devotion"（英文）
   - searchBG/searchSB 搜索英文原文和逐词
   - 找到匹配的英文词义 "devotion"
   - 从中文词义字段 `words_zh_fc` 中提取中文翻译（如 "奉爱"）
   - 返回搜索结果：`{ highlightKeyword: "奉爱", matchLocation: "wordmeaning", ... }`

2. **SearchResultList**：
   - 显示搜索结果，预览中显示对应的中文翻译 "奉爱"

3. **用户点击搜索结果**：
   - App.tsx 的 handleSearchResult 接收结果
   - 传递 `highlightKeyword: "奉爱"` 和 `matchLocation: "wordmeaning"` 到读页

4. **BGReadPage/SBReadPage**：
   - 接收 `highlightKeyword: "奉爱"` 和 `matchLocation: "wordmeaning"`
   - 传递给 SectionContent

5. **SectionContent**：
   - highlightText 函数使用 `highlightKeyword: "奉爱"`
   - 只在逐词释义位置高亮 "奉爱"
   - 自动滑动到第一个高亮位置

### 场景 2：中文模式下搜索中文关键词 "奉爱"

1. **SearchPage**：
   - 用户输入 "奉爱"（中文）
   - searchBG/searchSB 搜索中文字段
   - 找到匹配的中文词义 "奉爱"
   - 返回搜索结果：`{ highlightKeyword: "奉爱", matchLocation: "wordmeaning", ... }`

2. **后续流程**：与场景 1 相同

### 场景 3：中文模式下搜索英文关键词但中文缺失

1. **SearchPage**：
   - 用户输入 "xyz"（英文，但该节没有中文翻译）
   - searchBG/searchSB 搜索英文原文
   - 找到匹配但无对应中文翻译
   - 返回搜索结果：`{ highlightKeyword: "xyz", isMissingChinese: true, ... }`

2. **SearchResultList**：
   - 显示搜索结果，预览中显示 "[无中文翻译] xyz..."

3. **后续流程**：在读页高亮英文 "xyz"
