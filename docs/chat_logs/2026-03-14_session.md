1. 创建 `.github/workflows/deploy.yml` 工作流文件

3. 在 GitHub 仓库 Settings → Pages 中切换为 GitHub Actions 模式

**deploy.yml 关键内容：**
```yaml
name: Deploy to GitHub Pages
on:
  push:
    branches: [main]
jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: pnpm/action-setup@v4
      - uses: actions/setup-node@v4
        with:
          node-version: '20'
          cache: 'pnpm'
      - run: pnpm install
      - run: VITE_BASE_PATH=/vedabase-web/ pnpm build
      - uses: actions/deploy-pages@v4
```

**vite.config.ts 关键修改：**
```typescript
base: process.env.VITE_BASE_PATH || '/',
```

**遇到的问题：**

### 用户：数据加载不出来（GitHub Pages 上卡住）

**问题原因：**
数据请求路径硬编码为绝对路径（如 `/data/bg_data.json`），在 GitHub Pages 的子路径 `/vedabase-web/` 下会请求错误地址：
- 实际请求：`https://peterxie990624.github.io/data/bg_data.json`（404）
- 正确地址：`https://peterxie990624.github.io/vedabase-web/data/bg_data.json`

**修复方案：**
```typescript
// 修改前（错误）
const res = await fetch('/data/bg_data.json');

// 修改后（正确）
const BASE = import.meta.env.BASE_URL; // 开发时为 '/'，生产时为 '/vedabase-web/'
const res = await fetch(`${BASE}data/bg_data.json`);
```

### 用户：白边不好看，选中变白和浅色字混在一起

**问题分析：**
1. 章节列表项（`.list-item`）背景硬编码为 `white`
2. 选中/点击高亮颜色 `#ddeef8` 在浅色主题下太浅，文字看不清
3. 多个页面（BGChaptersPage、SBCantosPage、AkadasiPage、CalendarPage）有硬编码白色背景

**修复方案：**
```css
/* index.css - 修改前 */
.list-item {
  background: white;
}
.list-item:active {
  background: #ddeef8;
}

/* index.css - 修改后 */
.list-item {
  background: var(--veda-bg); /* 跟随主题变量 */
}
.list-item:active {
  background: #c8d8e8; /* 更深，文字清晰 */
}
```

**批量替换命令：**
```bash
sed -i "s/background: 'white'/background: 'var(--veda-bg)'/g" 文件名.tsx
```

### 用户：BG和SB深色模式导航栏都是白色

**问题原因：**
`TopNav` 组件依赖父组件传入 `theme` prop 来决定背景色，但 `BGChaptersPage`、`SBCantosPage` 等列表页面调用 TopNav 时没有传 `theme` 参数，导致默认使用浅色（白色背景）。

**修复方案：让 TopNav 直接通过 CSS 变量响应主题，不再依赖 JS prop**

```typescript
// TopNav.tsx - 修改后
// 移除对 theme prop 的依赖，改用 CSS 变量
const navStyle = {
  background: 'var(--veda-nav-bg)',      // 深色主题自动变深色
  color: 'var(--veda-nav-color)',         // 文字颜色跟随主题
  boxShadow: '0 1px 0 var(--veda-nav-border)',
};
```

```css
/* index.css - 浅色主题 */
:root, [data-veda-theme="light"] {
  --veda-nav-bg: #ffffff;
  --veda-nav-color: #1a2535;
  --veda-nav-border: rgba(0,0,0,0.08);
}

/* index.css - 深色主题 */
[data-veda-theme="dark"] {
  --veda-nav-bg: #1a2535;
  --veda-nav-color: #c8a84b;
  --veda-nav-border: rgba(255,255,255,0.08);
}
```

### 用户：国内访问慢、搜索进度条、开发调试面板

**问题1：国内访问慢**

根本原因：
- GitHub Pages 服务器在美国，国内访问延迟高
- 最大数据文件 `canto_10.json` 达 7.9MB，国内下载极慢

**解决方案：jsDelivr CDN**
- jsDelivr 在中国大陆有节点，专为 GitHub 仓库提供 CDN 加速
- URL 格式：`https://cdn.jsdelivr.net/gh/{用户名}/{仓库名}@{分支}/路径`
- 本项目：`https://cdn.jsdelivr.net/gh/peterxie990624/vedabase-web@main/client/public/data/`

**实现逻辑（useData.ts）：**
```typescript
// 自动检测 jsDelivr 可用性，优先使用，失败时回退 GitHub Pages
const JSDELIVR_BASE = 'https://cdn.jsdelivr.net/gh/peterxie990624/vedabase-web@main/client/public';
const GITHUB_BASE = `${import.meta.env.BASE_URL}`;

async function detectCDN(): Promise<'jsdelivr' | 'github'> {
  try {
    const controller = new AbortController();
    setTimeout(() => controller.abort(), 4000); // 4秒超时
    await fetch(`${JSDELIVR_BASE}/data/sb_index.json`, { 
      signal: controller.signal, method: 'HEAD' 
    });
    return 'jsdelivr';
  } catch {
    return 'github';
  }
}
```

## 本次对话涉及的关键技术知识点

### 1. GitHub Pages 部署静态网站

| 步骤 | 说明 |
|------|------|
| 创建工作流 | `.github/workflows/deploy.yml` |
| 配置 base 路径 | `vite.config.ts` 中 `base: process.env.VITE_BASE_PATH \|\| '/'` |
| 启用 Pages | Settings → Pages → Source: GitHub Actions |
| 触发部署 | 推送到 `main` 分支自动触发 |

### 2. Vite 环境变量

| 变量 | 说明 | 开发值 | 生产值 |
|------|------|--------|--------|
| `import.meta.env.BASE_URL` | base 路径 | `/` | `/vedabase-web/` |
| `import.meta.env.DEV` | 是否开发模式 | `true` | `false` |
| `import.meta.env.PROD` | 是否生产模式 | `false` | `true` |

### 3. CSS 变量主题系统

```css
/* 主题变量定义 */
:root { --veda-bg: #f5f0e8; }
[data-veda-theme="dark"] { --veda-bg: #0f1923; }

/* 使用 */
.element { background: var(--veda-bg); }
```

```typescript
// 切换主题
document.documentElement.setAttribute('data-veda-theme', 'dark');
```

### 4. jsDelivr CDN 加速 GitHub 内容

```
格式：https://cdn.jsdelivr.net/gh/{owner}/{repo}@{branch}/{path}
示例：https://cdn.jsdelivr.net/gh/peterxie990624/vedabase-web@main/client/public/data/bg_data.json
```

### 5. React 性能优化

- `useCallback`：缓存函数引用，避免子组件不必要的重渲染
- `useRef`：存储不触发重渲染的可变值（如 touchStartX）
- `useState` + `useEffect`：管理异步数据加载状态
