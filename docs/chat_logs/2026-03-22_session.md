## 开发实施记录

### 1. 会话记录更新

- 将 `2026-03-14_session.md` 中的所有待完成事项改为打钩状态（`- [x]`）
- 删除各节中的"学习要点"段落

### 2. 设置同步与 Toast 提示（BookshelfPage.tsx）

- 主页设置面板已支持语言、字号、主题切换，均有 Toast 提示
- 书架标题和书籍卡片标题随语言切换对调（中文主/英文副 ↔ 英文主/中文副）
- 设置通过 `useSettings` hook 持久化到 localStorage

### 3. 阅读页设置面板重构（TopNav.tsx）

- 将字号（A）、语言（EN/中）、书签三个独立按钮合并为一个设置（⚙️）下拉菜单
- 下拉菜单中所有选项均中英双语显示（如"语言 / Language"、"字号 / Font Size"）
- 切换语言和字号时均有 Toast 提示
- 新增目录（☰）按钮，位于设置按钮旁边
- 点击中央标题（如 Bg 1.7）也可打开目录
- 所有按钮添加按压反馈（`NavButton` 组件，按下时 `scale(0.88)` + 背景变色）

### 4. 目录面板（BGReadPage.tsx / SBReadPage.tsx）

- 从右侧滑入的目录面板，显示所有章节
- 当前章节展开显示所有节，当前节高亮并标记 ◀
- 点击章节标题跳转到该章第一节，点击节跳转到该节
- 点击遮罩层关闭目录

### 5. 触摸滑动灵敏度修复（BGReadPage.tsx / SBReadPage.tsx）

- 原逻辑：只判断水平位移 > 60px
- 新逻辑：水平位移 > 60px **且** `|dx|/|dy| > 1.5`（水平方向必须主导）
- 效果：斜向滑动不再触发翻页

### 6. 阅读进度记忆

- `BGReadPage`：每次阅读时将 `{chapterId, sectionIndex}` 存入 `localStorage['vedabase_progress_bg']`
- `SBReadPage`：同上，存入 `localStorage['vedabase_progress_sb']`
- `App.tsx handleSelectBook`：打开书籍时读取进度，若有则直接跳转到上次阅读位置

### 7. 导航返回逻辑修复（App.tsx）

- 从搜索结果进入阅读页（`fromSearch: true`），返回时切换到搜索 tab
- 从书签进入阅读页（`fromBookmark: true`），返回时切换到书签 tab
- 切换 tab 再回书架时，恢复之前的 `bookshelfRouteStack`（保留阅读位置）

### 8. 字体颜色统一（BGSectionsPage.tsx / SBSectionsPage.tsx）

- 节号（BG 1.1 等）：使用 muted 灰色（`#8aa0b4`），与章页一致
- 预览文字：使用主题文字色（`var(--veda-text)` / 深色模式 `#c0d0e0`）
- 不再使用金色（`--veda-blue` 在深色模式下是金色）

### 9. 全局字号联动（useSettings.ts / index.css）

- `useSettings` 在字号变化时将 `--veda-ui-font-size` CSS 变量写入 `html` 元素
- `index.css` 中 `.list-item` 和 `.canto-header` 使用 `var(--veda-ui-font-size)`
- 底部导航按钮添加 `:active { transform: scale(0.9) }` 按压反馈

### 10. 分享功能（SectionContent.tsx / BGReadPage.tsx / SBReadPage.tsx）

- 阅读页底部添加"复制经文 / Copy Verse"按钮
- 点击后将"节号 + 梵文原文 + 译文前200字 + 来源署名"复制到剪贴板
- 使用 Toast 提示复制成功/失败

### 11. 搜索历史全选/取消（SearchPage.tsx）

- 编辑模式下新增"全选"/"取消全选"按钮
- "完成"按钮在无选中时退出编辑模式，有选中时执行删除
- 删除按钮颜色在有选中时变为红色

### 12. 关于 App 翻译说明（AboutDialog.tsx）

- 已有翻译说明章节，明确说明：经典内容来自原版 App，界面文字由开发者（AI辅助）自行翻译
